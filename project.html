Delve Homes was the first serious project that I have worked upon while it is also a fun and simple website that was created by using the story The Machine Stops by E.M Forster where I had a team of people working with me to create a promotional website that would sell the underground living spaces where we were allowed creative freedom and this allowed us to be able to have fun with this project and to have a good time while exploring the ways that we have gathered through and the ways that we have thought of.
